#file:../copilot-commit-message-instructions.md

以下の情報（貼り付けられた差分/変更概要）から、コミットを安全に作るための提案をしてください。

要件:
- 分割が必要なら、ハンク/論理単位で **最大3件** までに分けて提案する（無理に分割しない）
- 各コミットについて「含める変更」と「コミットメッセージ案」をセットで出す（メッセージは上記ルール準拠）
- 必要なら `git add -p` 前提で「どの変更を同じコミットに入れるべきか」を短く説明する
- 実行コマンドは最後にまとめて提示してよい（ただし破壊的なコマンドは提案しない）

不足があれば最初に1〜3個だけ質問してから進めてください。
