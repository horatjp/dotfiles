# macOS 手動セットアップ手順

`setup.macos.sh` で自動化できない設定をまとめる。


## Finder

### デスクトップアイコン
- デスクトップで右クリック > 表示オプション
  - アイコンサイズ: **32×32**
  - グリッド間隔: 調整

### デスクトップアイコン画像変更
- アイコンを右クリック > 情報を見る > 画像をドロップ

### 拡張子の起動アプリ変更
- ファイルを右クリック > 「このアプリケーションで開く」 > その他
  - 「常にこのアプリケーションで開く」にチェック

### 右クリックで新規ファイル作成
**NewFile - Quick File Creation**（Mac App Store）を使用する。


## アプリ設定

### Raycast
- 起動ショートカット: `Option + Space`
- Clipboard History: `Control + V`（保存期間: 1ヶ月）
- アプリショートカット:
  | アプリ | ショートカット |
  |---|---|
  | 1Password | `Option + O` |
  | Finder | `Option + F` |
  | Google Chrome | `Option + C` |
  | Thunderbird | `Option + T` |
  | VS Code | `Option + V` |
  | WezTerm | `Option + W` |
  | システム設定 | `Option + S` |

### 1Password
- 設定 > 自動入力: **オフ**

### Google Chrome
- プロファイルを追加

### VS Code
- Setting Sync でログイン
- Project Manager 設定

### Thunderbird
- 証明書の例外を追加


### ミュージック

- ライブラリコピーをオフにする
  - ミュージック → 設定 → ファイル タブ
  - 「ライブラリに追加するときにファイルをコピー」のチェックを外す
